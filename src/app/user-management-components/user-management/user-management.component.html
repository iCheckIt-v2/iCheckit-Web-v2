Steps on how to import users

By following the given steps, web administrators can automatically import users into the web application.

  1. You can get a template or extract data using firebase-CLI
  2. Open terminal Make sure you are logged in using command: firebase login
  3. Type the command: firebase auth:export users.json --format=JSON
  4. Import the exported data using the import button
